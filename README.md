## GitHub Pages (consigliato)

Vedi **DEPLOY-GITHUB-PAGES.md** — attiva Pages in repo Settings, poi DNS CNAME su Serverplan.

```bash
# Anteprima locale
python -m http.server 8765
# http://localhost:8765/
```
